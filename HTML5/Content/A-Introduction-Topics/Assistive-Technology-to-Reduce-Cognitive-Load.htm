Assistive Technology to Reduce Cognitive Load

An accessible website not only broadens your audience but also ensures your organization’s message reaches everyone effectively. Not only that, but legally, there are certain standards you must adhere to.

Accessibility can only go so far – sometimes, assistive technology (ATs), a device or software that helps those with disabilities, is needed for a person to roam digitally and reduce cognitive load.

Video and audio ATs (1)(2)(3)

  • Video players that display captions. This can include transcripts of the video, captions, and subtitles that render any audio information from audio to text. This should include options for different languages.

    • Transcripts: A text version of the video.

    • Captions: synchronized text that appears on the screen while the video plays, which includes dialogue, speaker identification, and sound effects. Closed captions can be turned on or off, open captions cannot be turned off.

  • Software equalizers can be used to amplify or attenuate different frequency bands of an audio signal to improve the user’s perception. In other words, a user can regulate how loud audio is playing in one ear versus another ear.

  • Assistive listening devices can filter out sounds, or amplify sounds. This can be used to hear videos better and reduce background noise.

  • Noise-cancelling headphones can help create a controlled environment and reduce external sounds and reduce cognitive load that is related to sensory overload.

  • Screen readers can help reduce cognitive load by reducing the amount of energy associated with reading and staying focused, and help reduce distractions.
